Create a new video generation task using the Veo3 AI model.

## Usage Modes

### 1. Text-to-Video Generation
Provide a text prompt `prompt` to generate video content.

### 2. Image-to-Video Generation  
Provide a text prompt `prompt` and image URL list `imageUrls` to animate static images into videos.

<Note>
Image-to-video mode currently supports up to 2 images maximum.
</Note>
